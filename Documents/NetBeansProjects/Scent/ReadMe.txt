Add json library
Go to the project property
Go to Library
Add Library, then create new library and named it Gson
*All the files below are included in the project
Add gson-2.8.0.jar file to Classpath
Add gson-2.8.0-sources.jar to Sources
Add gson-2.8.0-javadoc.jar to Javadoc
Then click ok
Add the created Gson library  to the project
